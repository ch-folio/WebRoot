  1. W3Schools
    The largest web developer site on the Internet that makes learning to code easy for everyone.

    W3Schools offers an interactive learning experience that includes tutorials, illustrations, and a 'Try it Yourself' feature that allows for executing experimental code in real-time. It is a completely free resource but also offers a paid Pro User account, which gives access to even more resources such as video tutorials, website hosting, and no ads while browsing. An option to pay for web development certifications at the conclusion of the user's learning experiences makes it a great all-around choice for sharpening skills and building a portfolio.

    Rating: ✨ ✨ ✨




  2. MDN Web Docs
    MDN, previously known as the Mozilla Developer Network, is an open-source documentation repository with a vast amount of resources for devlopers and students.

    The aim of MDN is to assist developers in quickly and efficiently building projects for the open Web. More than 17 million monthly users browse and contribute to MDN, from developers to writers to other technologists. All users are free to contribute their knowledge to increase platform growth and users should also be on the lookout for educational additions from major partners of MDN such as Microsoft, Google, and Samsung.

    Rating: ✨ ✨ ✨ ✨ ✨




  3. LeetCode
    LeetCode is a large repository geared towards helping developers land the IT job of their dreams.

    LeetCode has assisted developers in getting hired by Facebook, Uber, Pinterest, Intel, Bank of America, Cisco and more. They manage this by offering nearly 3000 practice questions and challenges that can earn users rewards. The online assessments and training they offer helps developers prepare most specifically for the technical portions of their interviews. LeetCode keeps track of user progress while enhancing skills along the way, and they now offer support for a total of 14 of the most popular programming languages.

    Rating: ✨ ✨ ✨ ✨ ✨




  4. Code Wars
    Code Wars challenges users to strengthen their coding skills through small coding exercises called "kata".

    Challenges on this platform rank from beginner to expert levels and higher levels are reached as the user completes higher-ranked "kata" over time. There is an option to compare user solutions to the work of others, a pathway to train on specific skills sets is readily available and, best of all, Code Wars offers challenges for over 55 programming languages.

    Rating: ✨ ✨ ✨ ✨




  5. CSS - Tricks
    CSS - Tricks enhances developer skills with daily postings of articles covering CSS, HTML, JavaScript, and general web development.

    CSS - Tricks teaches developers how to dress up their code with crafty CSS additions. Their articles explain and share code with the user and oftentimes link to other relevant posts on various social media platforms for enhanced learning. Examples included in the article show the end result of the code they are sharing, and although navigation of the site can be confusing at times, it is a valuable resource when experimenting with code or looking to gain a new skill.

    Rating: ✨ ✨ ✨
